### Download

Download the latest `.dmg` file from the [GitHub releases page](https://github.com/saurabhav88/EnviousWispr/releases/latest/download/EnviousWispr.dmg) or from [enviouswispr.com](https://enviouswispr.com).

### Install

1. Open the downloaded `.dmg` file.
2. Drag EnviousWispr to your Applications folder.
3. Launch EnviousWispr from Applications.

### First launch

On first launch, macOS may show a security prompt because the app was downloaded from the internet. Click **Open** to proceed.

EnviousWispr runs as a menu bar app. Look for the lip icon in your menu bar after launching. There is no dock icon by default.

### Auto-updates

EnviousWispr includes automatic updates via Sparkle. You will be notified when a new version is available.
