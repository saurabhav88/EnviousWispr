# Run UI Test Scenarios — DEPRECATED

> **This skill is deprecated.** All behavioral testing is now handled by the UAT runner framework.
>
> - **For context-aware tests from git diffs**: use `wispr-run-smart-uat`
> - **For static regression suites**: use `wispr-run-uat`
> - **For ad-hoc AX inspection**: use `wispr-ui-ax-inspect`
> - **For CGEvent simulation**: use `wispr-ui-simulate-input`
> - **For screenshot verification**: use `wispr-ui-screenshot-verify`
>
> The legacy 6-step test flow and results interpretation table are documented in the
> [testing agent definition](../../agents/testing.md).
