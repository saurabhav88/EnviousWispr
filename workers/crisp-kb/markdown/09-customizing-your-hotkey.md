EnviousWispr lets you bind the recording trigger to almost any key combination, including modifier-only keys.

### Changing the hotkey

1. Open the settings window and go to the **Shortcuts** tab.
2. Click the hotkey recorder field.
3. Press the key combination you want to use.
4. The new hotkey is saved immediately.

### Supported keys

* Any standard key with or without modifiers (Cmd, Option, Control, Shift).
* **Modifier-only keys:** You can bind to a bare Option or Control key press. This is useful for quick access without reaching for a specific key.

### Separate hotkeys per mode

Push-to-Talk and Toggle modes each have their own hotkey binding. The active hotkey depends on which recording mode you have selected.

### How it works under the hood

EnviousWispr uses system-wide keyboard shortcuts that work globally in any app without requiring Accessibility permission for the trigger itself.
